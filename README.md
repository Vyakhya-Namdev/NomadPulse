# 🧭 NomadPulse

**NomadPulse** is a modern, full-stack web application inspired by **Airbnb** that allows users to explore and list vacation rentals around the world. 🌍  
Whether you're a traveler looking for a place to stay or a host wanting to list your property, NomadPulse offers a seamless and interactive experience.

## ✨ Features

🔐 **User Authentication**
- Register, login, and manage secure sessions using Passport.js

🏠 **Property Listings**
- Create, edit, and delete rental listings with title, description, price, and location

📷 **Image Uploads**
- Upload property images with Multer and store them using Cloudinary

🗺️ **Interactive Maps**
- View listings with real-time location on Mapbox-powered maps

📱 **Responsive Design**
- Fully responsive and mobile-friendly layout using Bootstrap + Custom CSS

🗄️ **Database Integration**
- Store and manage listing and user data efficiently with MongoDB

### 🔧 Backend
- **Node.js**
- **Express.js**
- **MongoDB** + **Mongoose** (for schema modeling)
- **Passport.js** (user authentication)
- **Multer** + **Cloudinary** (for image handling)

### 🎨 Frontend
- **EJS** (server-side templating)
- **Bootstrap** & **Custom CSS** (responsive UI)
- **Mapbox** (for interactive maps)
