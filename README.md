# 🧭 NomadPulse

**NomadPulse** is a modern full-stack web application inspired by Airbnb. It allows users to explore and list vacation rentals across the globe. The platform supports user authentication, listing management, image uploads, and map integration.

## 🌐 Live Demo

Check it out live here:  
👉 [https://nomadpulse.onrender.com/listings](https://nomadpulse.onrender.com/listings)

## 🚀 Features

- User authentication and session management
- Create, edit, and delete property listings
- Upload images using Multer and Cloudinary
- Mapbox integration for location display
- Responsive design with EJS and CSS
- MongoDB for database management

## 🛠️ Tech Stack

**Backend:**
- Node.js
- Express.js
- MongoDB & Mongoose
- Passport.js (authentication)
- Multer + Cloudinary (file upload)

**Frontend:**
- EJS (templating engine)
- Bootstrap & Custom CSS
- Mapbox (map integration)

**Deployment:**
- Hosted on [Render](https://render.com)
